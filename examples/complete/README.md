# Complete Example

## Overview
Shows all optional Site/Section/Subject/Story fields populated to demonstrate the full feature set.

## Structure
- Site with title
- Section with title and description
- Subject with title, description, and target component
- Three Story variations demonstrating different patterns

## Key Features
- Site title: Demonstrates top-level metadata
- Section title/description: Shows organizational grouping with metadata
- Subject title/description/target: Full component documentation setup
- Story variations:
  1. Minimal props-only (inherits target from Subject)
  2. Story with custom title and description
  3. Different props for same component
